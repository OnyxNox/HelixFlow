# HelixFlow

## Startup

1. Run docker compose detached from the repository's root directory, starting all of HelixFlow's services.
    ```shell
    docker compose up -d
    ```

## Shutdown

1. Run docker compose container shutdown from the repository's root directory.
    ```shell
    docker compose down
    ```
